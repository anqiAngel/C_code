C语言是结构化的程序设计语言！  
有三种结构:  
1. 顺序结构  
2. 选择结构  
3. 循环结构

---  
选择结构if-else if-else else和最近的if匹配
switch语句也是一种选择语句 常常用于多分支的情况 

    switch(整型表达式){
        语句项;
    }

    // case后是整型常量 不能是变量 字符型也可以 本质也是整型
    case 整型常量表达式:  
        语句;

    break跳出的是自己所在的switch语句

循环结构while
    
    while(条件){
        语句项;
        break;
        continue;
    }
    break直接跳出循环
    continue跳出本次循环
