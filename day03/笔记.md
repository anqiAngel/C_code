C语言提供的关键字  
1. C语言提供的,不能自己创建关键字  
2. 变量不能用关键字命名  
---
关键字:
1. auto用于修饰局部变量 自动创建,自动销毁的 -自动变量
auto int a = 100; 
auto 省略掉了一般  
auto 新的C语言语法中也有新的用法  
2. extern 是用来声明外部符号的
3. register 是寄存器关键字 register int num = 100; 建议num放入寄存器中存储 也不怎么用了  
4. signed和unsigned 有符号的和无符号的  
5. union联合体公用体 
define 和 include 都是预处理指令不是关键字
